# Product Analytics & User Retention Analysis

## Overview
Comprehensive product analytics platform tracking user engagement, retention funnels, feature adoption, and user behavior patterns to optimize product performance.

## Features
- **User Engagement Metrics**: Track DAU, MAU, session metrics
- **Retention Analysis**: Day-1, Day-7, Day-30 retention rates
- **Funnel Analysis**: User journey and drop-off tracking
- **Feature Adoption**: Monitor new feature adoption rates
- **Cohort Analysis**: Analyze user cohorts over time

## Project Structure
```
├── data/              # User events and behavior data
├── src/               # Analytics modules
├── notebooks/         # Product analytics notebooks
├── requirements.txt   # Dependencies
└── main.py           # Main analysis script
```

## Installation
```bash
pip install -r requirements.txt
```

## Usage
```bash
python main.py
```

## Key Metrics
- Daily Active Users (DAU)
- Monthly Active Users (MAU)
- Day-1/7/30 Retention Rate
- Feature Adoption Rate
- User Churn Rate
